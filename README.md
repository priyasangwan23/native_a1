# 📱 Smart Field Survey & Inspection App

> A modern React Native (Expo) mobile app for field surveys using Camera, Location, Contacts, and Clipboard APIs.

---

## 🚀 Features

### 🏠 Dashboard
- Welcome screen
- Today's survey count
- Quick action cards
- Recent survey summary

---

### 📝 Create Survey
- Site Name, Client Name
- Description, Priority, Date
- Form validation

---

### 📸 Camera Module
- Camera permission handling
- Capture & preview photo
- Capture timestamp
- Retake & delete (with confirmation)
- Loading indicator

---

### 📍 Location Module
- Get latitude & longitude
- Show accuracy
- Reverse geocoding (City, Region, Country)
- Map view with marker 🗺️
- Refresh location
- Copy location to clipboard

---

### 📞 Contacts Module
- Fetch device contacts
- Search contacts 🔍
- Contact counter
- Pull-to-refresh 🔄
- Avatar (initial-based)
- Copy contact number 📋
- "No Number" handling
- Empty state UI

---

### 📋 Clipboard Module
- Copy Survey ID
- Copy Contact Number
- Copy Location
- Paste notes
- Clear clipboard

---

### 📄 Survey Preview *(Extendable)*
- View all survey data
- Edit & submit survey

---

### 📚 Survey History *(Extendable)*
- View previous surveys
- Search & filter
- Delete with confirmation

---
